Account Actions

Options: lock account, unlock account, override, and terminate.

Lock AccountUnlock AccountOverrideTerminate

Lock Account

Locked accounts can only use account and support account ticket services. Requires staff.user.suspend.

Already Locked

This user has been locked for {Reason} until {Date of automatic unlock}.

Unlock Account

Clicking unlock immediately unlocks the account. Requires staff.user.suspend.revoke.

User Not Locked

The user is not locked. They were last locked {date} for {reason}.

Override

Displays active team/player suspensions and allows removing the suspension or overriding an organization's decision.

Terminate

Permanently bans and deletes account information tied to the account. Check Automatic Emails.
