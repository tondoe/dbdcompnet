Tournaments are used to create events. This frontend form covers the PDF requirements for event setup, playability, and publishing expectations.

Event Setup

Invite-Only events should let the organization choose registered teams and notify captains by automatic email later.

Publishing Flow

  • Create should publish the event on the public tournaments route and the organization subdomain route.
  • Invite-Only flows should notify selected captains by email from `noreply@dbdcompnet.com` after backend integration.
  • Editing should reopen all information fields for updates.
